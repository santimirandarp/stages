A step further: Top down idea

The previous list includes Huge Network, this is: The Cloud, the World Wide Web, or the internet. All of them are synonyms, and they are used in a lazy way, so be ready guess the meaning.

In a map, there are countries, cities, towns,... ; they themselves connected through highways, roads. For us, there are computers (cities or towns) connected by roads (links). The data (cars) flows on this roads. The Cloud or WWW is the whole thing, the map.

<--!

Conversely, we could think of the "physical" world of cities, and so on, as a network. A network is just a bunch of connected parts, called Nodes. Each node would be, for example, a city.

-->

This communication is between a server and a client. To keep the analogy, the server is the Market/Provider (provides a service), and you (your browser) is the client.

So the server and the client are 2 computers (cities or houses) communicating to each other. This is the most crazy idea here.

I will just turn it slightly crazier. Web servers (the market), and Web Browsers (you) are not the only type of connection between 2 computers. There are many others. But this is a topic for another post.

Last bits

Once a server is started on a computer, we can connect to it from home. A web server is basically a server that can be reached using a Web Browser. https://google.com is a web server, any newspaper site, any site! We always connect to this computer providing a service (server).

(Because a server can connect to other computers, a server can be hacked out. But it's not that easy.)

The most interesting part is that we can share information between 2 different computers (this is phones, PCs, etc.)

The page you're seeing right now is "hosted" in a server. It means that this is much like seeing a Word Document, but the file is actually in my -as opposed to your- computer!
